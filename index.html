Mabishu Revamp is based on Clearlooks Revamp GTK+ theme and elementary-monochrome icon set but with a lot of improvements to increase simplifications in your desktop.

Just transform your GNOME desktop in a more confortable work environment.

If your're running a GNOME-based desktop is very simple to install and user it. Just download the two tarballs from below and drop them into GNOME Appearance Dialog.

  • Download the GTK+ theme
  • Download the Icon theme
