Mabishu Revamps transforms your GNOME desktop in a more confortable work environment. A more consistent UI and almost integrated with the usual apps.

Mabishu Revamp is based on Clearlooks Revamp GTK+ theme and elementary-monochrome icon set but with a lot of improvements to increase simplifications in your desktop.

If your're running a GNOME-based desktop is very simple to install and use it. Just download the two packages from below and double clic onto them to install.

  • Download the GTK+ theme
  • Download the Icon theme
But if you want to enhance all your desktop install the nautilus-elementary improvements: 
sudo add-apt-repository ppa:am-monkeyd/nautilus-elementary-ppa
sudo apt-get update && sudo apt-get upgrade
